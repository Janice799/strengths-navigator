# StrengthsNavigator 구축 전략 보고서

## 1. 서론: 코칭 산업의 디지털 전환과 에이전트 주도 개발의 도래

현재 글로벌 라이프 코칭 및 경영자 코칭 시장은 파편화된 도구의 사용으로 인해 효율성이 저하되는 문제를 겪고 있습니다. 대다수의 강점(Strengths) 코치들은 클라이언트의 CliftonStrengths(구 StrengthsFinder) 34가지 테마 정보를 정적인 PDF 파일로 관리하며, 마케팅을 위해서는 Linktree, 뉴스레터 발송을 위해서는 Mailchimp, 일정 관리를 위해서는 Calendly와 같은 이질적인 플랫폼들을 수동으로 연결하여 사용하고 있습니다.

본 프로젝트는 Google Antigravity 플랫폼을 활용하여 구축될 예정입니다.

---

## 2. 퍼스널 브랜딩을 위한 동적 랜딩 페이지 구축 전략

### 2.1 강점 DNA의 시각화와 동적 프로필 구현

- **p5.js 라이브러리**를 활용한 제너레이티브 아트(Generative Art)
- 4가지 도메인(실행력, 영향력, 관계 구축, 전략적 사고)별 시각화
- 예: '전략적 사고' 도메인 → 깊이감 있는 프랙탈 구조, 블루 계열 패턴
- 예: '영향력' 도메인 → 확산되는 역동적인 오렌지/레드 계열 애니메이션

### 2.2 신뢰 구축을 위한 아티팩트(Artifacts) 배치와 전환 구조

- 증거 기반의 아티팩트 섹션 (인증 배지, 성공 사례)
- Barnum Effect를 활용한 마이크로 인터랙션 (성격 퀴즈 → 이메일 수집)
- CRM 연동 리드 관리

## 3. 상황 기반 맞춤 메시지 엔진: Contextual Strength Intelligence

### 3.1 34개 테마와 시공간 맥락의 매트릭스 설계

- **Gemini 2.5 Flash / Gemini 3 Flash** 모델 활용
- 1M 토큰 컨텍스트 윈도우

| 상황 | 강점 테마 | 도메인 | 메시지 컨셉 |
|------|----------|--------|-------------|
| 크리스마스 | 성취 (Achiever) | 실행력 | "올해 수많은 리스트를 지워낸 당신, 오늘 하루만큼은 '아무것도 하지 않기'를 목표로..." |
| 크리스마스 | 사교성 (Woo) | 영향력 | "당신의 에너지가 파티를 빛내겠지만, 오늘은 가장 소중한 한 사람과의 깊은 대화에..." |
| 여름 휴가 | 분석 (Analytical) | 전략적 사고 | "뇌과학적으로 3일 이상의 완전한 단절이 업무 효율을 20% 높입니다..." |

### 3.2 프롬프트 엔지니어링 및 페르소나 설정

- 코치 페르소나: '엄격한 트레이너형', '따뜻한 치유형', '논리적인 컨설턴트형'
- Thinking Config 활성화

## 4. 심미성 강화 및 소통 촉진을 위한 인터랙티브 아이디어

### 4.1 생성형 아트(Generative Art) 기반의 디지털 연하장
- react-p5 컴포넌트 사용
- 클라이언트 강점 조합에 따른 고유 패턴 생성
- '강점 만다라' 공유 기능 (바이럴 마케팅)

### 4.2 도파민을 자극하는 디지털 스크래치 카드
- react-scratchcard-v2 또는 Canvas API globalCompositeOperation
- "이번 주 강점 미션이 도착했습니다. 긁어서 확인해보세요!"

### 4.3 목표 달성을 축하하는 컨페티(Confetti)
- canvas-confetti 라이브러리
- 시즌별 커스터마이징 (눈송이, 별 모양)

### 4.4 3D 봉투 개봉 애니메이션
- CSS3 3D Transform (rotateX, transform-origin)
- 마이크로 인터랙션

## 5. CRM 기능 및 데이터 아키텍처 (Supabase 활용)

### 5.1 데이터베이스 스키마 설계

- `profiles` 테이블: 코치와 클라이언트의 기본 정보
- `strengths_definitions` 테이블: 34개 테마의 정의, 도메인, 설명
- `user_strengths` 테이블: 클라이언트와 강점 테마 매핑 (1~34위 순위)
- `interactions` 테이블: 메시지 발송 이력, 카드 오픈 여부, 메모
- `seasonal_prompts` 테이블: 상황별 프롬프트 템플릿

### 5.2 보안 및 실시간성

- RLS(Row Level Security) 기능
- Supabase Realtime 알림
